# Keskustelupalsta
Sovelluksessa käyttäjät voivat luoda tai liittyä eri keskuteluihin. Eri keskusteluaiheet on luokiteltu erikseen. Käyttäjillä on myös omat profiilisivunsa ja jokainen käyttäjä on joko peruskäyttäjä tai ylläpitäjä.

Sovelluksen ominaisuuksia:
- Sovelluksen etusivulla näytetään uusimmat keskustelut ja eri keskusteluaiheet
- Etusivulta käyttäjä voi kirjautua sisään tai luoda uuden käyttäjätunnuksen ja salasanan
- Käyttäjällä on oma profiili, joka sisältää tietoa käyttäjästä kuten nimen, profiilikuvan ja mahdollisesti muuta tietoa
- Käyttäjä voi luoda uuden keskusteluketjun haluamaansa keskusteluaihepalstaan
- Käyttäjä voi kommentoida eri keskuteluketjuissa
- Käyttäjä voi poistaa luomansa keskusteluketjun tai keskusteluketjuihin lähettämänsä viestit
- Keskusteluketjusta näkee milloin se on luotu ja viesteistä näkee milloin ne on lähetetty ja kuinka monta niitä on
- Käyttäjät voivat lähettää toisilleen myös yksityisiä viestejä
- Käyttäjä voi lähettää tai hyväksyä kaveripyyntöjä
- Ylläpitäjä pystyy samaan kuin käyttäjä, mutta voi myös poistaa muiden lisäämiä keskusteluita tai viestejä
